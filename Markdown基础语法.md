### 代码 {#代码}

`这是一个可以嵌入文本的代码块`

```python
class Solution(object):
    def twoSum(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: List[int]
        """
        for i in range(len(nums)):
            res=target-nums[i]
            if res in nums[i+1:]:
                return [i,nums[i+1:].index(res)+i+1]
```
